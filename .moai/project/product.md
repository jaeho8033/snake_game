# 제품 정의 - Snake Game

## 프로젝트 명
**Snake Game** (또는 **웹 스네이크**)

## 한 문장 설명
TypeScript와 HTML5 Canvas로 구현된 브라우저 기반 클래식 스네이크 게임으로, 현대적인 UI 제어와 다양한 게임플레이 옵션을 제공하며, 웹 브라우저에서 직접 실행 가능하고 링크 공유를 통해 쉽게 배포할 수 있는 웹 게임입니다.

## 대상 사용자
- 캐주얼한 웹 게임을 즐기는 사용자 (모든 연령대)
- JavaScript/TypeScript 학습자 및 개발자 (게임 개발 학습)
- 미니 게임 포트폴리오를 찾는 웹 개발자

## 핵심 기능 (MVP - 최소 기능 집합)

### 게임 메커니즘
- **뱀 이동**: 키보드 또는 화면 터치로 상/하/좌/우 방향 제어
- **음식 생성**: 게임판의 랜덤한 위치에 음식 자동 배치
- **충돌 감지**: 벽(게임판 테두리) 또는 뱀 자신과의 충돌 시 게임 오버
- **점수 표시**: 게임 진행 중 현재 점수 실시간 표시

### 게임 인터페이스
- HTML5 Canvas 기반 게임 렌더링
- 게임 시작/재시작 컨트롤
- 현재 점수 및 게임 상태(진행 중/게임 오버) 표시
- 난이도 선택 UI (낮음/중간/높음)
- 게임 진행 중 장애물 렌더링

## 확장 기능 (MVP 이상의 개선사항)

### 구현 완료
- **난이도 제어**: 사용자가 게임 시작 전 속도를 선택하는 레벨 옵션 (낮음/중간/높음) — SPEC-GAME-002 완료
- **벽 장애물**: 게임판 내부에 고정된 장애물을 배치하여 난이도 증가 — SPEC-GAME-002 완료

### 계획 중
- **색상 테마**: 라이트/다크 테마 등 게임 외관 커스터마이징 옵션 — SPEC-GAME-003 예정

## 주요 사용 사례

1. **캐주얼 게임플레이**: 사용자가 웹 브라우저에서 링크를 통해 게임에 접근하여 점수를 경쟁하며 즐김
2. **게임 개발 학습**: 초보 개발자가 실제 작동하는 게임 프로젝트를 분석하고 수정하며 학습
3. **빠른 배포 및 공유**: 빌드 후 정적 호스팅(GitHub Pages, Vercel 등)으로 즉시 배포 가능
4. **포트폴리오 프로젝트**: 웹 개발자가 자신의 기술 스택(TypeScript, Canvas)을 시연

```

  ├─ 뱀 이동 (방향 제어)

  └─ 점수 시스템

  ├─ 난이도 선택

```
